### 素质评分定位规则

#### 1. 根据素质项打分确定初始定位

首先根据各项素质的评分得到一个统计结果，例如：
（打低分的有几个素质项，打中高分的有几个素质项）
```javascript
{
  低分 素质项(对应1星) 数量: 1,    
  中低分 素质项(对应2星) 数量: 0,
  中分 素质项(对应3星) 数量: 1,  
  中高分 素质项(对应4星) 数量: 3,
  高分 素质项(对应5星) 数量: 0   
};
```
#### 1.1 高分情况（低分+中低分项 < 2个）
- **完全没有 低分 和 中低分**：
  - 若 (高分+中高分) 素质项 数量≥ 总素质项数×60% → 定位 x = "1"
  - 若 (高分+中高分) 素质项 数量 ≥ 总素质项数×40% → 定位 x = "2+"
  - 若 所有素质项 都是 中分 → 定位 x = "2"
  
- **仅有 1个低分 或 1个中低分** → 定位"2"

#### 1.2 低分情况（低分+中低分项 ≥ 2个）
- 若 (低分+中低分) 素质项 数量 ≥ 总素质项数×80% → 定位 x = "5"
- 若 (低分+中低分) 素质项 数量≥ 总素质项数×60% → 定位 x = "4"
- 若 (低分+中低分) 素质项 数量≥ 总素质项数×40% 或  (低分+中低分 = 2个) → 定位 x = "3"

#### 2. 根据（文化匹配度、求职意向）调整定位

在素质项最终定位 x 基础上：
- 若文化匹配度=1星 或 2星 → 定位降一档
- 若文化匹配度≥4星 且 求职意向≥4星 → 定位升一档
- 其他情况 → 保持 原定位 不变


## 新规则

素质项：不合格（1分）        待提升（2分）        胜任（3分）        优秀（4分）        卓越（5分）  
均分≤2   定位：不合格   
均分＞2 ≤2.5 定位：待提升     
＞ 2.5  ≤3.5之间  定位：胜任           
＞3.5  ≤4之间  定位：优秀       
均分＞ 4分 定位：卓越    
文化匹配度和求职意向不直接影响结果，仅供参考，最终定位可调整     
  
1、先求素质项均分，得出初始落位  
2、看单个素质项否有1 分项，若有，初始得分为待提升及以上，则落位待提升；初始定位为不合格，则落位不合格；若无，落位不变  
3、看是素质项否有2 分项，若有，初始得分为胜任及以上，则落位胜任；若初始定位为待提升，则落位待提升；若初始定位不合格，则落位不合格；若无，落位不变
